# Tetris

유니티를 처음 시작할 때 만들어본 테트리스 게임입니다.

<br/>

---

##  게임 소개

기본적인 테트리스를 똑같이 구현한 게임입니다.
이동, 회전, 줄 제거 등 테트리스의 기본적인 규칙을 그대로 구현했습니다.

<br/>

---

## 주요 기능

- 블록 생성 및 낙하 시스템
- 좌우 이동 및 회전 기능
- 라인 완성 시 제거 처리
- 게임 오버 처리

<br/>

---

## 플레이 목표

- 라인을 최대한 많이 제거하여 높은 점수 달성
- 블록이 쌓이지 않도록 효율적인 배치
